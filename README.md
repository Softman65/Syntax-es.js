# Syntax-es.js
Analizador sintactico de frases en castellano

el proposito de esta libreria es proporcionar un analisis sintactico para posteriores usos

## Example usage
``` Installation
```
``` js
const semantic = require('syntax-es.js')

const _syntax = new semantic('es')

console.log(_syntax.analisis('me pones 30 docenas de churritos fritos').morfologico() )
```
